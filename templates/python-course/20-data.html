Дата и время Python



Его имя datetime - это готовый модуль для определения текущей даты и выполнения действий над датами для представления даты на языке программирования Python.


import datetime

x = datetime.datetime.now()
print(x)


Результат:

2024-03-31 21:30:40.602510




Давайте объясним результат выше. Последовательность результата следующая. Дата включает год, месяц, день, часы, минуты, секунды и микросекунды. Есть много способов вернуть данные об объекте даты в модуле datetime. Мы сообщим вам день года и день недели в текущее время.


import datetime

x = datetime.datetime.now()

print(x.year)
print(x.strftime("%A"))


Результат:

2024
Tuesday




Создание объектов даты


Мы можем использовать класс модуля datetime() (конструктор) для создания даты. Для создания даты datetime(), классу требуются три параметра: год, месяц, день.


import datetime
x = datetime.datetime(2024, 3, 31)
print(x)


Результат:

2024-02-31 00:00:00




Функция strftime()


Объект datetime имеет метод форматирования объектов даты в читаемые строки. strftime() принимает один параметр для определения формата строки.


import datetime
x = datetime.datetime(2024, 3, 31)


Результат:

March





Информацию о коды фарматов:


  • %a - краткое название дней недели (Wed)
  • %A - полное название дня недели (Wednesday)
  • %w - в какой день недели, т.е. в какой день 0 - 6 (3)
  • %d - день месяца (31)
  • %b - краткое название месяца (Dec)
  • %B - полное название месяца (December)
  • %m - месяцы в цифрах 01 - 12 (12)
  • %y - короткое число года, последние две цифры (22)
  • %Y - полное название года (2024)
  • %H - час тарификация 00 - 23 (18)
  • %l - час описание 00 - 12 (09)
  • %p - час описание AM/PM (PM)
  • %M - минут описание 00 - 59 (35)
  • %S - секунда 00 - 59 (09)
  • %f - миллисекунда 000000-999999 (465845)
  • %Z - TimeZone (CST)
  • %j - день года 001 - 366 (263)
  • %C - выпуск века (21)
