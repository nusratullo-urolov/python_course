Классы и объекты Python



Язык программирования Python объектно-ориентированный язык программирования считается в котором почти все является объектом со своими свойствами и методами.



Создание класса (class)


Вы можете использовать вспомогательное слово class для создания класса. для примера приведем следующий пример.


class MyClass:
x = "Myblog IT"



Создание объекта


Мы можем использовать наш класс MyClass, который мы создали выше, для создания объекта.


p1 = MyClass()
print(p1.x)


Результат:

Myblog IT



Функция __init__()


Приведенные выше примеры не применяются в большинстве случаев, считая простейший класс и объект. Чтобы понять значение классов, нам необходимо сначала понять функцию __init__(). Все функции имеют функцию __init__(), которая всегда выполняется при запуске класса.


class Person:
def __init__(self, name, age):
self.name = name
self.age = age

p1 = Person("Muzaffar", 22)

print(p1.name)
print(p1.age)


Результат:

Muzaffar
22




Примечание: Функция __init__() вызывается автоматически каждый раз, когда вы используете класс для создания нового объекта.




Изменение параметров объекта


Вы можете тренировать параметры объекта следующим образом.


p1.age = 40



Удалить параметры объекта


del p1.age
