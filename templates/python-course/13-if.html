If...Else (Условный оператор) Python




Язык программирования Python поддерживает общие математические логические операции. Итак, это:


  • Равны: a == b
  • Не равны: a != b
  • Меньше: a < b
  • Меньше или равно a <= b
  • Больше: a > b
  • Больше или равно: a >= b


Вы можете процитировать вышеупомянутые термины в нескольких разных представлениях в одном условии. Я приведу пример для if условия в представлении ниже.


a = 13
b = 15
if b > a:
print("число b больше числа a ")


Результат:

число b больше числа a



Классификация приведенного выше примера заключается в том, что переменные a и b являются частью, необходимой для выполнения операции if. Условие сравнивает a и b в процессе выполнения если число b больше чем a условие выполняется, на экран число b больше числа a выводится слово.




Примечание: В языке программирования Python в условии if или других операциях перед оператором оставляется пробел для записи кода той части, которая является его частью. Помните: в языке программирования C++ if, for и т. д. мы будем использовать { } для записи операций!




Elif - "Иначе, если" переходит к этому условию, если предшествующее ему условие if принимает ложное значение. Не забывайте!!! elif всегда elif или if приходит перед использованием.


a = 13
b = 13
if b > a:
print("число b больше, чем a")
elif a == b:
print("a и b равны")


Результат:

a va b teng.



else - не проверяет, является ли какое-либо условие истинным ложным, давая значение "иначе". Else работает, если все вышеперечисленные условия принимают ложь. Не забывайте!!! он не приходит один, как elif.


a = 13
b = 15
if b == a:
print("Число b равно a")
else:
print("число b не равно a!")


Результат:

print("b soni a ga teng emas!")




IF краткая форма


Вам нужно будет написать только одну строку, если вы хотите использовать короткое тело условия if в действии условия.


if a > b: print("a больше, чем b")


Результат:

a больше, чем b



IF...ELSE краткая форма


Краткая форма т. е. if и else для формирования массива условие, что наша операция сначала напишем операторы, которые принимают истинное значение, а затем введем операторы, для которых выполняется else.


a = 13
b = 15
print("A") if a > b else print("B")


Результат:

B




AND


And на практике логика верна только тогда, когда оба условия принимают истинное значение.


a = 13
b = 15
c = 19
if a < b and c > a:
print("Оба условия приняли истинное значение!")


Результат:

оба условия приняли истинное значение!



OR


Or на практике логика верна только в том случае, если хотя бы одно из двух условий принимает истинное значение.


a = 13
b = 15
c = 19
if a < b and c < a:
print("Одно из двух условий принимает истинное значение!")


Результат:

Одно из двух условий принимает истинное значение!



PASS


If не может быть тела условия, если вы не хотите выполнять какие-либо действия внутри оператора условия тогда вы pass можете применить.


a = 13
b = 15
if b > a:
pass
