Строка Python



В языке программирования Python строки заключаются в одинарные или двойные кавычки. 'Salom' с "Salom" оба считаются одинаковыми. Вы можете использовать функцию print() вывода строки на черный экран.


print("Hello")
print('Hello')


Результат:

Hello
Hello




Создать переменную со строковым типом.


Чтобы прикрепить строковую строку к переменной, вы можете напрямую написать ее, не используя ключевые слова, которые не встречаются в языке программирования Python.


a = "Hello"
print(a)


Результат:

Hello




Присоединение многострочной строки к переменной.


Есть возможность заключить многострочную строку в 3 кавычки. обратите внимание на следующий пример.


a = """
Assalomu alaykum bugun siz bilan Python dasturlashini boshlaymiz.
Ustozinigiz yoki Yo'lboshchingiz.
Master Bekturdiyev
"""

print(a)


Результат:

Assalomu alaykum bugun siz bilan Python dasturlashini boshlaymiz.
Ustozinigiz yoki Yo'lboshchingiz.
Master Bekturdiyev




Строки считаются массивами.


В языке программирования Python, как и во многих других языках программирования, строки считаются состоящими из байтов в выражении. Однако в языке программирования Python нет символьных типов данных, вы просто принимаете длину одного символа как 1. [ ] в квадратных скобках представлены данные.


a = "Hello, World!"
print(a[1])


Чтобы обрезать определенную часть строки и загрузить ее в какую-либо переменную или распечатать на экране, мы пишем начало и конец в квадратных скобках, а двоеточие отделяется друг от друга точкой. [начало: конец]


b = "Hello, World!"
print(b[2:5])


Результат:

llo




Для обратной обрезки "-" используем. При этом индекс последнего символа записывается первым.


b = "Hello, World!"
print(b[-5:-2])


Результат:

orl




Используйте функцию len(), чтобы получить длину строки .


a = "Hello, World!"
print(len(a))


Результат:

13




Используйте функцию lower(), чтобы сделать все элементы строки маленькими.


a = "Hello, World!"
print(a.lower())


Результат:

hello, world!




Используйте функцию upper(), чтобы сделать все элементы строки большими.


a = "Hello, World!"
print(a.upper())


Результат:

HELLO, WORLD!
