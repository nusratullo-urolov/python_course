Модуль Python



Создание модуля можно рассматривать как то же самое, что и библиотека языка программирования Python. Вкратце, мы можем вычислить файл, состоящий из нескольких функций. Модуль можно использовать для его вызова, поместив необходимый набор кодов в отдельный файл, чтобы создать его.




Примечание: Если мы приравняем создание модуля к уровню библиотеки языка программирования, у вас будет возможность создать библиотеку, создав свой собственный модуль.




devsmodul.py


def greeting(name):
print("Hello, " + name)


вызов модуля считается таким же, как вызов библиотеки.


import devsmodul

mymodule.greeting("Qudrat")




Переменные в модуле


Модуль может содержать функции, как описано выше, а также все типы переменных (массивы, словари, объекты и т. д.):


devsmodul.py


person1 = {
"name": "Muzaffar",
"age": 22,
"country": "Uzbekistan"
}



import devsmodul

a = mymodule.person1["age"]
print(a)




Переименование модуля


Вы можете создать псевдоним при импорте модуля, используя ключевое слово as


import devsmodul as mx
a = mx.person1["age"]
print(a)




Использование функции dir()


Существует встроенная функция для перечисления всех имен функций (или имен переменных) в модуле. Функция dir():


import platform

x = dir(platform)
print(x)


Результат:

['DEV_NULL', '_UNIXCONFDIR', 'WIN32_CLIENT_RELEASES', 'WIN32_SERVER_RELEASES']['DEV_NULL', '_UNIXCONFDIR', 'WIN32_CLIENT_RELEASES', 'WIN32_SERVER_RELEASES']




Примечание: функция dir() может использоваться в всех модулях, а также в модулях, которые вы создаете самостоятельно.




Импорт из модуля


Используя ключевое слово from, вы можете импортировать только детали из модуля. или вы можете ввести символ * после ключевого слова import, чтобы вызвать все части модуля.


mymodule.py


def greeting(name):
print("Hello, " + name)

person1 = {
"name": "Muzaffar",
"age": 22,
"country": "Xorazm"
}


мы вызываем часть, содержащую mymodule, упомянутую выше.


from mymodule import person1
print (person1["age"]) #25
