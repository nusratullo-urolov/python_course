JSON Python





Язык программирования Python содержит элементы JSON и модуль для работы над ним для этого достаточно вызвать следующее.


import json



Чтение JSON (text) в Python


Если у вас есть строка JSON, вы можете проанализировать ее с помощью функции json.loads().


import json

x = '{ "name":"Muzaffar", "age":22, "city":"Qo'shko'pir"}'
y = json.loads(x)
print(y["age"])


Результат:

22




Преобразование Python JSON в массив JSON


Если у вас есть объект Python, вы можете преобразовать его в массив JSON с помощью функции json.dumps().


import json

x = {
"name": "Muzaffar",
"age": 22,
"city": "Qo'shko'pir"
}

y = json.dumps(x)

print(y)


Результат:

{ "name":"Muzaffar", "age":22, "city":"Qo'shko'pir"}



Вы можете преобразовать следующие типы объектов Python в строки JSON:


  • dict
  • list
  • tuple
  • string
  • int
  • float
  • True
  • False
  • None
