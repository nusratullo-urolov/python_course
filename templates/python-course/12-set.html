Set (Множество) Python



Язык программирования Python имеет четыре набора данных.

Set - это неупорядоченное и неиндексированное множество. В Python коллекции записываются фигурными скобками.


thisset = ("apple", "banana", "cherry")
print(thisset)




Примечание: Наборы неупорядочены, поэтому вы не можете быть уверены, в каком порядке будут отображаться элементы.




Элементы ввода.


Вы не можете получить доступ к множеством с индексом или ключевым словом. Но вы можете с помощью tsikl просматривать элементы.


thisset = {"apple", "banana", "cherry"}
for x in thisset:
print(x)


Результат:

cherry
banana
apple



Чтобы проверить, есть ли элемент в set, мы можем применить следующий путь.


thisset = {"apple", "banana", "cherry"}
print("banana" in thisset)


Результат:

True



После создания коллекции вы не можете изменять ее элементы, но можете добавлять новые.


Используйте метод add(), чтобы добавить один элемент в множество Вы можете использовать функцию update(), если хотите добавить несколько элементов в коллекцию.



thisset = {"apple", "banana", "cherry"}
thisset.add("orange")
print(thisset)


Результат:

{'apple', 'orange', 'cherry', 'banana'}



thisset = {"apple", "banana", "cherry"}
thisset.update(["orange", "mango", "grapes"])
print(thisset)


Результат:

{'orange', 'apple', 'grapes', 'cherry', 'mango', 'banana'}





Мы используем функцию len(), чтобы определить, сколько элементов в множество.


thisset = {"apple", "banana", "cherry"}
print(len(thisset))


Результат:

3



Чтобы удалить элемент из набора, используйте метод remove() или discard().


thisset = {"apple", "banana", "cherry"}
thisset.remove("banana")
print(thisset)


Результат:

{'cherry', 'apple'}




Примечание: Если удаляемый элемент недоступен, remove() возникает ошибка. discard() ошибка в функции не возникает.




Удалите "банан" следующим способом discard():


thisset = {"apple", "banana", "cherry"}
thisset.discard("banana")
print(thisset)


Результат:

{'cherry', 'apple'}



Вы можете pop() использовать его для удаления элемента, но этот метод удалит последний элемент. Помните, что множество беспорядочные, поэтому вы не будете знать, какой элемент будет удален. Возвращаемое значение метода pop() это удаленный элемент.


thisset = {"apple", "banana", "cherry"}
x = thisset.pop()
print(x) #удаленные состояние
print(thisset) # предыдущее состояние


Результат:

banana
{'cherry', 'apple'}



В языке программирования Python есть несколько способов добавить два или более множество(set). Вы можете использовать такие функции, как union() и update().




Примечание: union() и update() исключает любые дубликаты данных.




set1 = {"a", "b" , "c"}
set2 = {1, 2, 3}

set1.update(set2)
print(set1)


Результат:

{1, 2, 3, 'b', 'c', 'a'}



set() использует конструктор для создания соответствующего set множество.


thisset = set(("apple", "banana", "cherry"))
print(thisset)


Результат:

{'banana', 'cherry', 'apple'}




Python имеет встроенный набор функций, которые вы можете использовать в пакетах.
