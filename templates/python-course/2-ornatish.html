Установка Python



В моей статье об установке языка программирования Python, \я постараюсь дать вам краткое изложение. В наши дни интернет для нас не проблема, поскольку мы возможность найти в интернете то, что нам нужно. В частности, мы будем использовать интернет непосредственно в процессе установки среды Python. Я опишу процесс установки шаг за шагом. Итак, давайте начнем процесс установки.


Язык программирования Python является переменным языком. Последняя версия Python 3.7.4 вышла (3.09.2019). Поймите, какая именно версия вам нужна в начале. есть изменения для каждой версии некоторые области кода, которые вы скомпилировали в python2.7, или библиотеки, которые вы использовали, имеют другой тон для версии, которую вы только что установили, или, может быть, он использовал другую библиотеку в качестве новой технологии вместо библиотеки.


Шаг 1. Скачать Python.


Официальный сайт языка программирования python войдите в python.org и нажмите Downloads. При этом он предлагает вам библиотеки для операционных систем. теперь мы собираемся установить язык программирования python для WINDOWS. сейчас python-3.6.1-amd64.скачиваем exe.



Шаг 2. Python-3.6.1-amd64.exe файл.



Загруженный файл python окажется в папке Downloads на вашем компьютере. Дважды щелкните по нему, чтобы установить.



Шаг 3. python-3.6.1-amd64.установить exe.




Щелкните команду Install Now и не забудьте выбрать команду Add Python 3.6 to PATH не забудьте. Таким образом, следующие окна будут выглядеть следующим образом.




Мы успешно установили теперь с вами, чтобы убедиться, что среда Python работает правильно. для этого нажимаем START+R. пишем CMD. К полученному стеклу



Результат:

> python36



напишем. окно формируется следующим образом, если вы установили его правильно.
