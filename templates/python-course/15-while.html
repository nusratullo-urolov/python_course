While (Цикл) Python



В Python есть две команды цикла:


  • Циклы while
  • Петли for


Циклы while


В цикле while сначала проверяется условие затем запускается тело цикла. мы можем проиллюстрировать это следующим образом.


i = 1
while i < 6:
print(i)
i += 1




Примечание: не забудьте увеличить i, иначе цикл будет продолжаться без остановок.




Break в while


Оператор break гарантирует, что этот цикл не будет выполнять шаги, следующие за этой командой.


i = 1
while i < 6:
print(i)
if i == 3:
break
i += 1
