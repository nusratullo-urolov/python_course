List (Список) Python



Язык программирования Python имеет четыре набора данных:


  • List - это набор, который можно изменить. Допускает повторных участников.
  • Tuple - это инвариантное множество. Допускает повторных участников.
  • Set - это набор, который не отсортирован и не пересмотрен. Нет повторяющихся членов.
  • Dictionary - это неупорядоченный, изменяемый и индексируемый набор. Нет повторяющихся членов.

List - Набор данных.




List - это набор, который можно изменить. В Python списки записываются в квадратных скобках.




thislist = ["apple", "banana", "cherry"]
print(thislist)


Результат:

['apple', 'banana', 'cherry']




Ссылка на индексный номер


Вы можете получить доступ к элементам списка, обратившись к номеру индекса:


thislist = ["apple", "banana", "cherry"]
print(thislist[1])


Результат:

banana




Обращение к номеру обратного индекса.


Обратная индексация относится к концу до начала -1, последнему предложению, -2 второму последнему предложению и так далее.


thislist = ["apple", "banana", "cherry"]
print(thislist[-1])


Результат:

cherry




Диапазон индексов.


Вы можете указать диапазон индексов, указав, где они должны начинаться, а где заканчиваться. При определении диапазона возвращаемое значение образует новый список с указанными элементами.


thislist = ["apple", "banana", "cherry", "orange", "kiwi", "melon", "mango"]
print(thislist[2:5])


Результат:

['cherry', 'orange', 'kiwi']




Примечание: поиск начинается с индекса 2 (включительно) и заканчивается индексом 5 (не включается).




Проверка значения из списка.


Используйте ключевое слово, чтобы определить, присутствует ли выделенный элемент в списке in.


thislist = ["apple", "banana", "cherry"]
if "apple" in thislist:
print("Yes, 'apple' is in the fruits list")


Результат:

Yes, 'apple' is in the fruits list




Длина списка.


List используйте эту функцию, чтобы определить, сколько элементов доступно в len():


thislist = ["apple", "banana", "cherry"]
print(len(thislist))


Результат:

3




Добавление элементов.


Чтобы добавить что-то в конец списка append(), используйте метод:


thislist = ["apple", "banana", "cherry"]
thislist.append("orange")
print(len(thislist))


Результат:

['apple', 'banana', 'cherry', 'orange']




Удаление элемента.


Есть несколько способов удалить элементы из списка: remove() метод очищает указанный объект:


thislist = ["apple", "banana", "cherry"]
thislist.append("banana")
print(len(thislist))


Результат:

['apple', 'cherry']



Метод pop() удаляет указанный каталог (или последний элемент, если индекс не указан) :


thislist = ["apple", "banana", "cherry"]
thislist.pop()
print(len(thislist))


Результат:

['apple', 'banana']



Ключевое слово del() указанный каталог будет удален:


thislist = ["apple", "banana", "cherry"]
del.thislist[0]
print(len(thislist))


Результат:

['banana', 'cherry']




Примечание: Вы также можете удалить список с помощью ключевых слов del навсегда:




Метод clear() очищает список:


thislist = ["apple", "banana", "cherry"]
thislist.clear()
print(len(thislist))


Результат:

[ ]




Скопируйте список.


Потому что вы не можете просто скопировать список записей list2 = list1: list2 только для записей, которые умирают list1, и list1 внесенные изменения также автоматически list2. Есть способы копирования, Один из них - использовать встроенный метод списка copy()..


thislist = ["apple", "banana", "cherry"]
mylist = thislist.copy()
print(mylist)


Результат:

['apple', 'banana', 'cherry']




Добавление двух списков.


В Python есть несколько способов объединить или связать два или более списков. Один из самых простых способов + использовать оператор .


list1 = ["a", "b", "c"]
list2 = [1, 2, 3]

list3 = list1 + list2
print(list3)


Другой способ добавить два списка - добавить все элементы из списка 2 в list1 один за другим.


list1 = ["a", "b", "c"]
list2 = [1, 2, 3]

for x in list2:
list1.append(x)

print(list1)


Или вы можете использовать метод extend() для добавления элементов из одного списка в другой список: Используйте метод extend() для добавления list2 в конец 1-го списка :


list1 = ["a", "b" , "c"]
list2 = [1, 2, 3]

list1.extend(list2)
print(list1)
