Массив Python




Массив - это специальная переменная, которая может содержать несколько значений одновременно. Например:


cars = ["Ford", "Volvo", "BMW"]




Чтение элементов массива


В процессе чтения элементов массива достаточно обратиться к индексу.


print(cars[0])


Результат:

Ford




Примечание: элементы массива индексируются от 0.




Длина массива


Длину массива можно найти с помощью вспомогательной функции len()


x = len(cars)


Результат:

3



Отдельное чтение элементов массива


Вы можете распечатать элементы массива отдельно, используя команду for in.


for x in cars:
print(x)


Результат:

Ford
Volvo
BMW



Добавление элемента в массив


Вы можете использовать метод append() для добавления элемента в массив. Пример ниже сосредоточьтесь на.


cars.append("UzAvto ;(")


Результат:

Ford
Volvo
BMW
UzAvto ;(



Удаление элемента в массив.


Вы можете использовать вспомогательные функции pop() и remove()


cars.pop(1)
cars.remove("UzAvto ;(")
