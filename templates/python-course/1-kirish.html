Введение в Python



Python - один из широко используемых в настоящее время языков программирования.


Автор: Гвидо ван Россум (Gvido van Rossum).


Разработано: 1991 году


Официальный сайт: python.org


Расширения файлов: .py, .pyc, .pyd, .pyo, .pyw, .pyz.


Посмотрите ниже. Это фреймворки Python, используемые в различных областях программирования.


  • Веб-программирования: Django, Pyramid, Bottle, Tornado, Flask, web2py
  • Интерфейс программирования: TkInter, PyGObject, PyQt, PySide, Kivy, wxPython
  • Scientific and Numeric: SciPy, Pandas, IPython
  • Компьютерные Программы: Buildbot, Trac, Roundup
  • Администратор Сервера: Ansible, Salt, OpenStack



Пачему Python?


  • Python работает на разных платформах. (Windows, Linux, Mac и т.д.)
  • Python имеет простой синтаксис, похожий на английский.
  • Язык программирования Python позволяет программисту писать меньше кода, чем другие языки программирования.
  • В Python есть возможность создавать веб-приложения, настольные и мобильные приложения.
  • Python библиотеки намного больше! вы делаете почти всю свою работу через библиотеки!
  • Python имеет гораздо более высокую скорость обработки, занимая меньше места в памяти! PyPI-сайт библиотек Python!



Наибольший потенциал языка программирования Python заключается в его открытом коде и обилии библиотек, а задано и в компактности содержащегося в нем кода. прямо сейчас мы предлагаем вам написать слово "Hello, World" на 3 языках программирования: C, Java и Python!



Из этого видно, что на языке программирования Python код, вводимый программистом, сжат и занимает меньше вашего времени.


Для работы языка программирования Python не требуется мощный компьютер. Использование кода, который вы пишете, на нетбуке, компьютере с более простой характеристикой, возможно даже на вашем Android-смартфоне в руке.


Считается очень хорошим языком программирования для сервера. в настоящее время среди программистов существует так называемый язык программирования, созданный для хакеров.
