Числа Python



В языке программирования Python существует 3 различных типа чисел, и переменные числового типа создаются, когда вы присваиваете им значение.


  • int
  • float
  • complex


x = 1 # int
y = 2.8 # float
z = 1j # complex




Язык программирования Python type() имеет возможность определять тип входящей переменной или фиксированной переменной, используя ее функцию для определения типа любой переменной.


x = 1
y = 2.8
z = 1j

print(type(x))
print(type(y))
print(type(z))


Результат:

< class 'int' >
< class 'float' >
< class 'complex' >




int - это соответствующие числа, которые являются десятичными числами которые не имеют остатка от целого значения и включают отрицательные или положительные числа.


x = 1
y = 254646213354488
z = -5586928

print(type(x))
print(type(y))
print(type(z))


Результат:

< class 'int' >
< class 'int' >
< class 'int' >




float - отрицательные или положительные числа, соответствующие приведенному выше правилу (int), но содержащие десятичные дроби.


x = 1.10
y = 1.0
z = -32.5

print(type(x))
print(type(y))
print(type(z))


Результат:

< class 'float' >
< class 'float' >
< class 'float' >




complex - Комплексные числа записываются как мнимая часть "j".


x = 3 + 5j
y = 6j
z = -5j

print(type(x))
print(type(y))
print(type(z))


Результат:

< class 'complex' >
< class 'complex' >
< class 'complex' >




Язык программирования Python имеет возможность преобразовывать числовые типы из одного типа в другой, как и другие языки программирования. Для этого можно использовать int(), float() и complex() функции.


# конвертировать целое число в float
x = float(1)

# преобразование из float в int
y = int(2.8)

# преобразование из int в complex
z = complex(x)

print(x)
print(y)
print(z)


Результат:

1.0
2
(1+0j)
< class 'float' > < class 'int' > < class 'complex' >




Случайное число.


У Python random() нет функции генерации случайных чисел , но у Python random есть встроенный модуль, который можно использовать для генерации случайных чисел :


import random

print(random().randrange(1, 10))


Результат:

9
