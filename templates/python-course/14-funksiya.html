Функция Python



Функция - это просто блок кода, который запускается во время вызова. В качестве параметров в функцию можно передавать данные. Функции используются для выполнения определенной задачи. У вас есть возможность использовать функцию, которую вы создали в процессе написания кода, один или несколько раз или вообще не использовать ее.


Создание функции


В языке программирования Python для создания функции используется ключевое слово def. Например:


def my_function():
print("Hello Myblog IT")




Вызов функции


При вызове функции мы используем ее имя, если параметры, мы используем параметры, если в противном случае мы используем пустую скобку ().


def my_function():
print("Hello Myblog IT")

my_function()


Результат:

Hello Myblog IT




Аргументы


Аргументы, которым будет присвоена функция, должны быть указаны в скобках рядом с именем функции. при этом у вас будет возможность передать указанные аргументы функции. В приведенном ниже примере мы создадим функцию, которая извлекает имя из аргумента name, добавляя слово "Developer".


def my_function(name):
print(name + " Developer")

my_function("Farrukh")
my_function("Qudrat")
my_function("Muzaffar")


Результат:

Farrukh Developer
Qudrat Developer
Muzaffar Developer




Аргументы часто сводятся к аргументам в документации Python.




Python Math



В языке программирования Python есть готовый математический модуль и функциональная работа для выполнения операций это имя модуля math. Это поможет вам выполнить цифровые шаги. Мы рассмотрим с вами несколько математических функций.


min() и max()


Используется для нахождения наименьшего и наибольшего из чисел.


x = min(5, 10, 25)
y = max(5, 10, 25)

print(x)
print(y)


Результат:

5
25



abs()


Дает положительное значение заданного значения. Он возвращает позитив, даже если он положительный.


x = abs(-7.25)

print(x)


Результат:

7.25



pow(x, y)


Функция вычисляет x степени y. При этом у вас будет возможность подсчитать числа под корнем.


x = pow(4, 3)

print(x)


Результат:

64



sqrt()


Функция выдает заданное значение в качестве подсчета под корнем второго порядка.


import math

x = math.sqrt(64)
print(x)


Результат:

8



ceil()


Функция приближает данное значение к ближайшему целому числу вверх.


import math

x = math.ceil(1.5)
print(x) # 2


Результат:

2



floor()


Функция приближает данное значение к ближайшему целому числу вниз.


import math

x = math.floor(1.4)
print(x) # 1


Результат:

1
