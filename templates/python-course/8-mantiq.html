Boolean (Логика) Python



Что общего с большинством популярных сегодня языков программирования, так это логические выражения. На языке программирования мы с вами знаем истину (True) и ложь (False). True или False - это значения, возвращаемые при сравнении двух значений. Например:


print(10 > 9)
print(10 == 9)
print(10 < 9)


Результат:

True
False
False



Если вы инициализируете оператор условия IF, вам будет важно, вернут ли вам значения True или False и это то, что мы увидим в следующем примере.


a = 200
b = 33

if b > a:
print("b больше, чем a")
else:
print("b меньше, чем a")



Результат:

b больше, чем a




У вас есть возможность оценить значение на языке программирования Python все, что вам нужно сделать, это использовать функцию bool().




print(bool("Hello"))
print(bool(15))


Результат:

True
True




В этом случае любое число возвращает значение True кроме 0. В строках каждая непустая строка возвращает значение True.


bool("abc")
bool(123)
bool(["apple", "cherry", "banana"])


Результат:

True
True
True




Давайте посмотрим на некоторые процессы, которые возвращают значение False. Обратите внимания, что если заданы пустые значения ( ), [ ], { }, " ", из чисел 0, None, он вернет False


bool(False)
bool(None)
bool(0)
bool("")
bool(())
bool([])
bool({})
