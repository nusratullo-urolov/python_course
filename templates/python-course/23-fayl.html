Файл Python




В языке программирования Python есть несколько шагов, которые вы можете выполнить над файлами. Создавать, читать, обновлять и удалять.



Работа с файлами


Основной функцией работы с файлами является функция open (). Эта функция принимает два параметра file_name и file_режим.



Способы открытия файлов


  1. "r" - Read - значение по умолчанию. Открывает файл для чтения, ошибка, если файл не существует;
  2. "a" - Add - открывает файл для добавления, если он недоступен, создает его;
  3. "w" - Write - открывает файл для записи, если он недоступен, создает его;
  4. "x" - Create - создает указанный файл, возвращает ошибку, если файл существует;


Вы также можете указать, должен ли файл работать в двоичном или текстовом режиме.



  • "t" - Текст-значение по умолчанию. Текстовый режим
  • "b" - Двоичный-двоичный режим (например, изображения)


Если вы хотите открыть файл, не помечая его в каком-либо режиме, вы запишете само имя файла. в этом случае по умолчанию будет "R", то есть для открытия файла.


f = open("demofile.txt")



Открыть файл другими способами то есть открыть его с другой целью.


f = open("demofile.txt", "rt")





Чтение файла Python



Используйте функцию open() для открытия файлов. Функция read() используется для чтения файла. в приведенном ниже примере функция read будет работать с перенаправлением на open() (это всегда будет случай).


f = open("demofile.txt", "r")
print(f.read())



Если читаемый файл находится в другом месте, необходимо указать его местоположение перед именем file_name.


f = open("D:\\myfiles\welcome.txt", "r")
print(f.read())



Файл читается до n символов.


Чтобы прочитать файл до определенного символа, все, что нам нужно сделать, это передать количество символов функции read().


f = open("demofile.txt", "r")
print(f.read(5))



Чтение строк


Вы можете вернуть одну строку, используя функцию readline():


f = open("demofile.txt", "r")
print(f.readline())



Используя readline() функцию двойного умножения, вы можете прочитать две первые строки:


f = open("demofile.txt", "r")
print(f.readline())
print(f.readline())



Через цикл строк файла вы можете читать весь файл строка за строкой:


f = open("demofile.txt", "r")
for x in f:
print(x)



Закрыть файлы


Когда вы закончите, рекомендуется всегда закрывать файл.


f = open("demofile.txt", "r")
print(f.readline())
f.close()




Примечание: Вы всегда должны закрывать свои файлы, в некоторых случаях из-за буферизации изменения, внесенные в файл, могут не отображаться, пока вы не закроете файл.





Написание / создание файла Python



Запись в существующий файл


Вам нужно добавить параметр в функцию open(), чтобы записать ее в существующий файл:


  • "a" - Append - добавляется в конец файла;
  • "w" - Write - переписывает любой существующий контент.


f = open("demofile2.txt", "a")
f.write("Now the file has more content!")
f.close()

f = open("demofile2.txt", "r")
print(f.read())



Создать новый файл


Чтобы создать новый файл в Pythonopen(), используйте один из следующих параметров open():


  • "x" - Create - создает файл, возвращает ошибку, если файл существует;
  • "a" - Add - создает файл, если указанный файл недоступен;
  • "w" - Write - создает файл, если указанный файл недоступен.


myfile.txt fayl yaratish.


f = open("myfile.txt", "x")



Результат: создан новый пустой файл!


  • Если его нет, создайте новый файл:

f = open("myfile.txt", "w")





Удалить файл Python

Мы будем использовать модуль os для удаления файла на языке программирования Python. Для запуска удаления.необходимо использовать функцию os.remove().


import os
os.remove("demofile.txt")



Проверка наличия файла:


Чтобы избежать ошибки, вы можете проверить наличие файла перед его удалением:


import os
if os.path.exists("demofile.txt"):
os.remove("demofile.txt")
else:
print("Файл не найден")



Удалить папку


Следующая для удаления всей папки используйте функцию os.rmdir():


import os
os.rmdir("myfolder")
