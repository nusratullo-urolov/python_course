Tuple (Кортеж) Python



Язык программирования Python имеет четыре набора данных.


Tuple - это инвариантное множество. Допускает повторных участников.


thistuple = ("apple", "banana", "cherry")
print(thistuple)



Вы можете получить доступ к элементам кортежа, указав номер индекса в квадратных скобках. Процесс индексирования начинается с индекса 0, как и большинство объектно-ориентированных языков программирования.


thistuple = ("apple", "banana", "cherry")
print(thistuple[1])


Результат:

banana



К нему также можно обратиться через обратный индекс в квадратных скобках. См.пример ниже для примера. В процессе обратной индексации последний элемент равен -1.


thistuple = ("apple", "banana", "cherry")
print(thistuple[-1])


Результат:

cherry




Диапазон индексов


Вы можете указать диапазон индексов, указав, где они должны начинаться, а где заканчиваться. При определении диапазона возвращаемое значение образует новый список с указанными элементами.


thislist = ("apple", "banana", "cherry", "orange", "kiwi", "melon", "mango")
print(thislist[2:5])


Результат:

('cherry', 'orange', 'kiwi')




Изменение значений кортежа.


После создания tuple вы не можете изменить его значения. Кортежи неизменяемы. Но есть обходной путь. Вы можете преобразовать ячейку в список (List), изменить список и перевернуть список.


x = ("apple", "banana", "cherry")
y = list(x)
y[1] = "kiwi"
x = tuple(y)

print(x)


Результат:

('apple', 'kiwi', 'cherry')




Вывод значений tuple через цикл.


Для этого нам нужно, чтобы вы могли просматривать элементы с помощью цикла for.


thistuple = ("apple", "banana", "cherry")
for x in thistuple:
print(x)


Результат:

apple
banana
cherry




Проверить наличие элемента.


Ключевое слово in используется для определения наличия указанного элемента в ячейке.


thistuple = ("apple", "banana", "cherry")
if "apple" in thistuple:
print("Да, 'apple' существует внутри кортежа.")


Результат:

Да, 'apple' существует внутри кортежа.



Используйте следующий метод len(), чтобы определить, сколько элементов в кортеж


thistuple = ("apple", "banana", "cherry")
print(len(thistuple))


Результат:

3




Удаление объектов


Кортежи неизменяемы, поэтому вы не можете удалить из него элементы, но вы можете полностью удалить элемент внутри:


thistuple = ("apple", "banana", "cherry")
del.thistuple
print(thistuple)




Примечения: Ошибка возникает при объявлении удаленной переменной кортежа.




Результат:

Traceback (most recent call last):
File "demo_tuple_del.py", line 3, in < module >
print(thistuple) #this will raise an error because the tuple no longer exists
NameError: name 'thistuple' is not defined




Добавление переменных.


Чтобы добавить две кортежные переменные, выполняется действие + (add).


tuple1 = ("a", "b", "c")
tuple2 = (1, 2, 3)

tuple3 = tuple1 + tuple2
print(tuple3)


Результат:

('a', 'b', 'c', 1, 2, 3)




tuple() конструктор.


мы можем реализовать, используя конструктор tuple() для преобразования в тип кортежа.


thistuple = tuple(("apple", "banana", "cherry"))
print(thistuple)


Результат:

('apple', 'banana', 'cherry')
