Наследование (Класс/Class) Python



Наследование позволяет нам определить класс, который наследует все значения из другого класса. При этом классы называются родительским классом и дочерним классом.


  • Родительский класс: это считается унаследованным классом, который также называется базовым классом.
  • Дочерний класс: это класс, унаследованный от другого класса, также называемый производным классом.



Создание родительского класса


Любой созданный класс может быть родительским классом, поэтому синтаксис считается таким же, как и при создании других классов.


class Person:
def __init__(self, fname, lname):
self.firstname = fname
self.lastname = lname

def printname(self):
print(self.firstname, self.lastname)

x = Person("John", "Doe")
x.printname()



Создание дочернего класса


Чтобы создать класс, который наследует функциональность от другого класса, отправьте базовый класс в качестве параметра при создании дочернего класса.


class Student(Person):
pass




Примечание: Вы не хотите добавлять какие-либо другие свойства или методы в класс, используйте ключевое слово pass. Он останавливает выполнение всего элемента (внутри класса) после этого слова и останавливает действие текущего класса.




Теперь у Student есть то же свойство, что и у класса Person.


x = Student("Muzaffar", "Bekturdiyev")
x.printname()



Добавить функцию __init__()


Выше мы создали дочерний класс с вами, который наследует родительские свойства, мы пишем функцию __init__() вместо ключевого слова pass.


class Student(Person):
def __init__(self, fname, lname):
# несколько команд




Примечание: добавление функции __init__() не наследует функцию __init__() в родительском классе.




Чтобы сохранить функцию родительского класса, нам нужно будет прикрепить функцию родительского класса __init__() к дочернему классу.


class Student(Person):
def __init__(self, fname, lname):
Person.__init__(self, fname, lname)


Используйте функцию super()


В языке программирования Python также есть функция, которая наследует все методы и свойства дочернего класса super() от родительского класса.


class Student(Person):
def __init__(self, fname, lname):
super().__init__(fname, lname)
