Try Except Python



В языке программирования Python рекомендуется использовать try except, когда заранее неизвестно, что код вызывает ошибку, или когда требуется проверка. Если вы не прикрепите этот код случай ошибки кода программа будет вынуждена остановиться автоматически.


  1. try - Позволяет тестировать блок кода в программном коде на наличие ошибок.
  2. except - Программный код позволяет устранить ошибку.
  3. else - Позволяет выполнять код, когда в программном коде нет ошибок.
  4. finally - Позволяет выполнять код независимо от результата тестирования в программном коде - и вне блоков.


try:
print(x)
except:
print("если здесь нет x")



Много исключений


Вы можете указать столько блоков исключений, сколько захотите, например, если вы хотите выполнить специальный блок кода для конкретной ошибки:



try:
print(x)
except NameError:
print("X не найден")
except:
print("еще одна ошибка")
